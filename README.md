TraceLog
================================

1. About the TraceLog
--------------------------------------
The TraceLog is a utility project for operating on application traces.

2. Dependencies
--------------------------------------
- JRE 8
- Maven 3.x
- Lombok


3. Building
--------------------------------------

``
mvn clean compile package
``

--------------------------------------
